# StudyHub Cafe - Big Data Analytics Project

## Overview
StudyHub Cafe is a Big Data Analytics and Machine Learning project developed to analyze customer behavior, booking trends, revenue forecasting, and business performance for a smart study cafe business in Nilai, Malaysia.

## Features
- Big Data Analytics
- Machine Learning Regression
- Predictive Analytics
- Interactive Dashboard
- Data Visualization
- Customer Behavior Analysis
- Revenue Forecasting

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn
- XGBoost

## Dataset
- 5500+ booking records
- Data collected from 2020–2024
- Includes customer behavior, bookings, ratings, and revenue data

## Machine Learning Models
- Linear Regression
- Random Forest Regression
- XGBoost Regression

Linear Regression achieved the best predictive performance, while XGBoost was additionally used for feature importance analysis.

## Visualizations
- Revenue Trend Analysis
- Booking Heatmap
- Customer Rating Distribution
- Payment Method Analysis
- Feature Importance Analysis
- Residual Analysis
- Future Revenue Forecasting
